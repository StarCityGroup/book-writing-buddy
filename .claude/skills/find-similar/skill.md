# Find Similar Content

Find similar or duplicate content to detect redundancy or check for unintentional plagiarism.

## Parameters

- `text` (string, required): Text to find similarities for
- `threshold` (float, optional): Similarity threshold 0-1 (default: 0.85)
- `limit` (integer, optional): Max results (default: 10)

## Returns

List of similar text chunks with similarity scores and source information.
