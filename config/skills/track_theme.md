# Track Theme

**Name:** `track_theme`

**Description:** Follow a concept or theme across all chapters in the manuscript

**Parameters:**
- `theme` (str, required): Theme, concept, or keyword to track

**Workflow Steps:**

1. Use `find_cross_chapter_themes` to search for the theme across all chapters
2. For each chapter where theme appears:
   - Use `search_research` with the theme keyword to find specific mentions
   - Note the context and how the theme is discussed
3. Identify patterns in how the theme evolves across chapters
4. Note connections and variations in treatment
5. Suggest opportunities for stronger thematic connections or consistency

**Example Usage:**
- "Track the theme 'resilience' across all chapters"
- "Where does 'infrastructure failure' appear in the book?"
- "Find mentions of 'adaptation trap' across chapters"
